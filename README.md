# 🧠 Prisoner's Dilemma Strategy Simulator

A Game Theory simulator built with Python and Gradio.

## Features
- 🤖 AI vs AI mode with 6 strategies
- 🧑 Human vs AI mode
- 🏆 Full Tournament (all strategies vs each other)
- 📊 Live score charts with Matplotlib

## Strategies Included
- Always Cooperate
- Always Betray
- Random
- Tit-for-Tat
- Mostly Cooperate
- Mostly Betray

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Interact with the Gradio app

## Built With
- Python
- Gradio
- Matplotlib

## Concepts
Game Theory | Strategic Decision Making | AI Simulation
